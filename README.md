# warp-started

1- Create project with warp 

```
$ rustup update stable
$ cargo --version
cargo 1.45.0 (744bd1fbb 2020-06-15)
$ cargo new warp-started
$ cd warp-started/
$ cargo build  
$ cargo run 
```

2- Test HTTP Server using Terminal

```
$ curl http://localhost:3030/hello/warp
Hello, warp!

```
